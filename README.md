# Piano 88 Keys - 寿司打スタイル音ゲー 

88鍵盤ピアノアプリに寿司打風のコースシステムを搭載！

##  主な機能

###  寿司打コースシステム
- **3000円コース** (初級): きらきら星、カエルの歌、ちょうちょう
- **5000円コース** (中級): ジングルベル、ハッピーバースデー、メリーさんの羊、エリーゼのために
- **10000円コース** (上級): All I Want for Christmas、天国と地獄、トルコ行進曲、ラカンパネラ

###  自動演奏モード
- ボタンで自動演奏開始/停止
- ボタンで曲スキップ（全13曲）
- 寿司エフェクト付き！

###  C++ネイティブオーディオ
- 超低レイテンシ音声生成
- Taylor展開による高速sin計算
- ADSRエンベロープ処理
- 5倍音合成でリッチな音色

##  動作環境
- Android 8.0以上
- APKサイズ: 23.2MB
- デジタルサイネージ対応

##  技術スタック
- Flutter SDK
- C++17 (ネイティブオーディオ生成)
- CMake build system
- JNI (Java Native Interface)
- Method Channel (Flutter-Native通信)
